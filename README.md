# Beauty Wudhu Game

Game edukasi React tentang kosmetik waterproof, validitas wudhu, dan rutinitas muslimah yang seimbang.

## Run

```bash
npm install
npm run dev
```

Lalu buka URL Vite yang muncul, biasanya `http://localhost:5173`.

## Stack

- React
- Vite
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Fitur

- Pilih karakter: student, office worker, influencer
- Pilih beauty kit maksimal 3 produk
- Prayer time events
- Score halal, beauty, ilmu, dan balance
- Badge dan multiple endings
- Sound effects berbasis WebAudio
- Layout responsive mobile
